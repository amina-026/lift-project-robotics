# lift-project-robotics
 Arduino Elevator System

A simple elevator automation system built using Arduino and C++.

 Extended Description

This project simulates a basic elevator system using Arduino. The system utilizes an ultrasonic sensor to detect the current floor based on distance measurements, and push buttons to select the desired floor. An L298N motor driver controls the movement of the elevator (up and down).

The project demonstrates fundamental concepts of embedded systems, including sensor integration, motor control, and real-time decision-making. It also reflects problem-solving skills in designing a simple automation system.

 Features
Floor detection using ultrasonic sensor
Button-based floor selection (3 floors)
Motor control using L298N driver
Automatic movement (up/down) based on target floor
Serial monitoring for debugging
 Technologies Used
Arduino (C/C++)
Ultrasonic Sensor (HC-SR04)
L298N Motor Driver
Embedded Systems Basics
 How It Works
The ultrasonic sensor measures distance to determine the current floor
User selects a floor using buttons
System compares current floor with target floor
Motor moves elevator up or down accordingly
Stops automatically when target floor is reached
 Project Type
